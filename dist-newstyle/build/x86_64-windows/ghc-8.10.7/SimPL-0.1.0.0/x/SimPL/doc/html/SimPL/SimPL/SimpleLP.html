eval :: Expr -> LookUpTable -> IO (Maybe Int) #

Given a LookUpTable evaluates an expression. It can return Nothing for unassigned lookups, division and mod operations. This is context dependent! (depends on the LookUpTable used)

exec :: Instr -> LookUpTable -> IO LookUpTable #

Given a LookUpTable, executes an instruction. It returns a new LookUpTable. It can return the same LookUpTable if an instruction equivalent to the NoOp (or the NoOp) is executed
